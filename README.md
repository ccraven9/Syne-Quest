# SyneQuest

A 2D adventure game built with Godot!  


## 🧩 Tools

- **Engine:** Godot Engine 4.6.1


## 📦 Requirements

Before getting started, make sure you have the following installed:

- **Godot Engine 4.6.1**
  - Download from: https://godotengine.org/download

- (Optional) VS Code
  - VS Code recommended with the **Godot Tools** extension
